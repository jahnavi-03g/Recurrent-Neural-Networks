# Recurrent Neural Networks (RNN)

A hands-on implementation and exploration of Recurrent Neural Networks using Python and Jupyter Notebook.

## 📌 Overview

This notebook covers the fundamentals of RNNs — a class of neural networks designed to work with sequential data such as time series, text, and speech. It walks through the architecture, forward/backward pass through time (BPTT), and practical implementation using deep learning frameworks.

## 🧠 Topics Covered

- What are Recurrent Neural Networks?
- Vanishing and exploding gradient problem
- LSTM (Long Short-Term Memory) and GRU (Gated Recurrent Unit)
- Sequence modeling with real data
- Training and evaluation

## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- TensorFlow / Keras (or PyTorch)
- NumPy, Matplotlib
